# 🔒 Proyecto: Encriptador y Desencriptador 🔓

<CalloutBox type="info">
  Este proyecto fue desarrollado como parte de un desafío de **Alura Latam**, donde implementé un encriptador y desencriptador de textos utilizando **HTML**, **CSS** y **JavaScript**. El objetivo era practicar lógica de programación y manipulación del DOM.
</CalloutBox>

## 💡 ¿Qué es el Encriptador y Desencriptador?

El **Encriptador y Desencriptador** es una aplicación web que permite a los usuarios ingresar un texto, encriptarlo utilizando un conjunto de reglas simples, y desencriptarlo de vuelta. Este proyecto es útil para aprender sobre algoritmos básicos de transformación de texto y la manipulación de entradas de usuario.
