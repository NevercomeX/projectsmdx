# 🎮 Proyecto: Juego de El Ahorcado 🎮

<CalloutBox type="info">
  Este proyecto fue creado para implementar el clásico juego de **El Ahorcado** utilizando **HTML**, **CSS**, y **JavaScript**. Mi objetivo fue desarrollar una experiencia de juego interactiva y amigable para el usuario, mejorando mis habilidades en la manipulación del DOM y la lógica de programación.
</CalloutBox>

## 💡 ¿Qué es El Ahorcado?

El **Ahorcado** es un juego donde los jugadores intentan adivinar una palabra antes de que el dibujo de un personaje ahorcado esté completo. Es ideal para poner a prueba tanto la lógica del jugador como las habilidades del desarrollador para manejar eventos del usuario y actualizar dinámicamente la interfaz.
